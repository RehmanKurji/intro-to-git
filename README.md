# GIT Tutorial

### Initialize

git init

### Adding Files to GIT
 
git status 

git add <FILENAME>

git status

### Committing to Git

git commit -m "<MESSAGE>"

### View History

git log

### Add All Files 

git add -A 

OR

git add .

### Add File of Certain Type
git add *.<FILETYPE EXTENSION (ie html)>

### Remove File (before commit)

git reset HEAD <FILENAME>

### Listing Branches

git branch

### Create Branch

git checkout -b <BRANCH NAME>

### Switching Branches

git checkout <BRANCH NAME>

### Merging Branches

git merge <BRANCH NAME>
